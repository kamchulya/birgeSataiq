# БіргеСатайық — платформа для партнёрских продаж

Продавайте больше — вместе. Подключайте продажников к своему магазину, выдавайте реферальные ссылки, автоматически распределяйте комиссии.

## 🎯 Что это?

Платформа, где продавцы могут привлекать продажников (амбассадоров), которые получают процент с каждой продажи по своей реферальной ссылке.

## 🛠 Технологии

- **Frontend**: HTML/CSS/JS
- **Backend**: Supabase (PostgreSQL + Edge Functions)
- **Hosting**: Vercel
- **Payments**: Zammler (эскроу)
